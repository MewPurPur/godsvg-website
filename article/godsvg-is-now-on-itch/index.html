GodSVG is now available on Itch

By: MewPurPur Sep 2, 2024
NOTE:
This blog post was written retroactively to document events from GodSVG's early development history.

Hello everyone!

Along with the release of GodSVG's fifth alpha, I have another announcement to make.

GodSVG on Itch.io

I have recently published GodSVG on Itch.io, where you can download it or use it directly from the browser. Users are politely encouraged by Itch.io to purchase GodSVG for $10, but it is available for free. Itch.io will get the same version of GodSVG as all other platforms (alpha 5 currently) and there won't be any delays.

This means that you can now get GodSVG officially on three different places:

  • Github: Downloads
  • godsvg.com: Downloads and Web editor
  • Itch.io: Downloads & Web editor

While I’m unfamiliar with digital distribution platforms, I’ve always admired Itch.io, and I thoroughly researched it to make sure I did everything correctly, and that my project is appropriate for it. If you find any issues, please let me know.

Support

Many thanks to everyone who donates to this project! Donations have allowed me to dedicate a lot more time to GodSVG while balancing other responsibilities, and without them, the project would be far behind where it is today. If you find GodSVG valuable and are able to donate, I would be very grateful!
